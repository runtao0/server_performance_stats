# server_performance_stats
## Practice bash scripting project

### Inspiration/Guide
Using [roadmap.io DevOps roadmap "Server Performance States" guided project](https://roadmap.sh/projects/server-stats)

### MVP Requirements (from website)
* Total CPU usage

* Total memory usage (Free vs Used including percentage)

* Total disk usage (Free vs Used including percentage)

* Top 5 processes by CPU usage

* Top 5 processes by memory usage
